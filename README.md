# Storyline
Android application which creates souvenirs (stories) from user's trips effortlessly.

App flow:
* Create a story
* Travel
* Get a souvenir

Main Features:
* Build a route on a map
* Associate images to locations
* Classify images (Machine Learning)
* Create a Souvenir (GIF)
